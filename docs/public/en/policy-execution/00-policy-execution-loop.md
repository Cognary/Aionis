# Policy and Execution Loop

Aionis closes the loop from memory to governed action.

## Loop Components

1. Rule evaluation: `rules/evaluate`
2. Tool selection: `tools/select`
3. Decision recording: `tools/decision`
4. Feedback learning: `tools/feedback`

## Lifecycle and Promotion

1. Rule lifecycle from draft to active
2. Shadow evaluation before active rollout
3. Adaptation under explicit policy controls
