# Roadmap

The roadmap focuses on product-facing capability evolution.

## Near-Term

1. Stronger context orchestration presets for domain workloads
2. Expanded benchmark packs with reproducible reports
3. Deeper production observability and gate automation

## Mid-Term

1. More planner-native policy contracts
2. Stronger cross-runtime integration patterns (MCP, LangGraph, ecosystem SDKs)
3. Enhanced replay and governance workflows

## Long-Term

1. Strategy memory modules for long-horizon agents
2. Richer multi-layer context composition
3. Higher-level memory operating model for enterprise teams
