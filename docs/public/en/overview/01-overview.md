# Overview

Aionis is a `Memory Kernel` for production agent systems.

It is designed as executable infrastructure, not just a retrieval layer:

1. Memory that is persisted and queryable.
2. Policy that evaluates and constrains behavior.
3. Action that is selected, traced, and replayable.

## Category Position

1. Aionis is not a plain vector database.
2. Aionis is not a context-loader-only utility.
3. Aionis is not a workflow shell.
4. Aionis is a memory kernel with verifiable state and operational controls.

## Core Loop

`Memory -> Policy -> Action`

1. Memory: `write / recall / context assemble`
2. Policy: `rules/evaluate`
3. Action: `tools/select + tools/decision + tools/feedback`
