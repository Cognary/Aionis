# Role-Based Reading Paths

Use these paths to reach the right depth quickly.

## 3-Minute Path (New User)

1. [Overview](/public/en/overview/01-overview)
2. [5-Minute Onboarding](/public/en/getting-started/02-onboarding-5min)
3. [Aionis Onepage](/public/en/about/02-aionis-onepage)

Outcome:

1. Understand category, value, and first successful API loop.

## 10-Minute Path (Engineer)

1. [Get Started](/public/en/getting-started/01-get-started)
2. [Build Memory Workflows](/public/en/guides/01-build-memory)
3. [API Reference](/public/en/api-reference/00-api-reference)
4. [SDK Guide](/public/en/reference/05-sdk)

Outcome:

1. Run write/recall/context/policy calls in a real integration.

## 30-Minute Path (Architect)

1. [Core Concepts](/public/en/core-concepts/00-core-concepts)
2. [Architecture](/public/en/architecture/01-architecture)
3. [Context Orchestration](/public/en/context-orchestration/00-context-orchestration)
4. [Policy and Execution Loop](/public/en/policy-execution/00-policy-execution-loop)
5. [Operate and Production](/public/en/operate-production/00-operate-production)
6. [Production Core Gate](/public/en/operations/03-production-core-gate)

Outcome:

1. Understand system boundaries, production model, and governance surface.

## Page Dedup Rules

1. `Overview / Core Concepts / Architecture / Context Orchestration` are primary pages.
2. `about/*` and `concepts/*` pages are supplements and should link back to primary pages.
3. Prefer primary pages for onboarding and external product communication.
