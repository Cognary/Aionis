# Aionis Onepage

Last updated: `2026-03-03`

## What Aionis Is

Aionis is a `Verifiable / Operable Memory Kernel` for agent systems.

It turns memory from passive storage into executable infrastructure:

1. Durable memory graph writes
2. Policy-aware decision routing
3. Auditable and replayable execution traces

## Problems It Solves

1. Long-lived memory across sessions and workflows
2. Reliable recall for LLM-ready context assembly
3. Governed behavior via rules and feedback loops
4. Production operation with gates, diagnostics, and benchmarks

## Core Architecture

1. System of record: graph objects + commit lineage
2. Derived async pipeline for embeddings and other artifacts
3. Context assembly and policy execution as explicit API contracts
4. URI-first inspect/replay for operational debugging

## Capability Surface

1. Memory APIs: `write / recall / recall_text / context/assemble / find / resolve`
2. Policy loop APIs: `rules/evaluate / tools/select / tools/decision / tools/feedback`
3. Session/event ingestion APIs for interaction memory
4. SDKs for TypeScript and Python
5. Deployable in standalone and production service topologies

## Operational Confidence

1. Production gates for consistency and SLO checks
2. Replayable run/decision lineage for incident review
3. Benchmark snapshots with reproducible commands
4. Tenant/scope isolation and access controls
