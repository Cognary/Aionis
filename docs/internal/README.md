# Internal Docs

This directory is reserved for local/internal planning and remediation documents.

It must not be published to the public docs site.

## Structure

1. `internal/remediation`: audit/security/remediation records.
2. `internal/strategy`: commercial/release/packaging strategy working docs.
3. `internal/plans`: engineering implementation plans and migration plans.
