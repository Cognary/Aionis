# Aionis Blog Site

Independent Astro blog for Aionis.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

Default ports:

- `4322` for development
- `4323` for preview

## Deployment

Set `BLOG_SITE_ORIGIN` for canonical URLs and sitemap generation.

Example:

```bash
BLOG_SITE_ORIGIN=https://blog.aionisos.com npm run build
```
