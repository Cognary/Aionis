Use this section for task-based, production-oriented walkthroughs.

## What this section is for

1. Follow an end-to-end path instead of reading API pages in isolation.
2. Copy a working sequence and adapt it to your stack.
3. Build confidence for integration, replay, release, and policy tuning.

## Before you start

1. Have a running Aionis environment.
2. Know your `tenant_id`, `scope`, and auth mode.
3. Be ready to keep `request_id`, `run_id`, `decision_id`, and `commit_uri` from the examples.
